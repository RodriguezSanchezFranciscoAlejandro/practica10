Inicio
Define una función llamada reverse_string que toma una cadena como argumento.
Dentro de la función reverse_string, convierte la cadena en un array de caracteres usando el método split("").
Luego, invierte el orden de los elementos en el array usando el método reverse().
Finalmente, une los elementos del array invertido en una cadena nuevamente usando el método join("") y devuelve la cadena invertida.
Define una cadena de ejemplo, por ejemplo, "Hello, World!".
Muestra la cadena original en la consola.
Llama a la función reverse_string con la cadena original como argumento para obtener la cadena revertida.
Muestra la cadena revertida en la consola.
